### Demo Video: `streamlit_demo.mp4`

**Location:** `demo/` folder

**Description:**
This video demonstrates the **TherapyX AI Mental Health Assistant** interacting with users via the **Streamlit interface**. It highlights the AI’s real-time message analysis and response features.

**Video Highlights:**

1. **User Interaction:**

   * A user types messages into the Streamlit interface.
   * The AI processes each message instantly.

2. **AI Response:**

   * Harmful messages are detected and flagged.
   * Regular messages are responded to appropriately.

3. **Interface Features:**

   * Streamlit UI displays the messages in real time.
   * Shows AI’s risk assessment and responses directly on the dashboard.

**What You’ll See in the Video:**

* Live typing and message submission in the Streamlit app.
* Real-time detection of harmful or risky content.
* AI response displayed instantly within the interface.

**Purpose:**
The video demonstrates the **Streamlit interface in action**, showing how users can interact with TherapyX and receive **instant AI feedback**, making the system intuitive, responsive, and user-friendly.
